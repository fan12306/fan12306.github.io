    构建服务器 http

  • http帮忙创建编写服务器

  • http.createServer() server.on('request' , funciton() {})但客户端请求发过来，就会自动触发服务器的 request 请求事件，然后执行第二个参数，回调处理

  • server.on('request' , funciton() {})`

    • request 请求事件处理函数，需要接受两个函数；
      • Requset请求对象
        • 请求对象可以用来获取客户端的一些请求信息
      • Response
        • 响应对象可以用来给客户端发送响应消息
        • write方法可以用来给客户端发送响应数据，write可以书写多次但是必须要end来结束响应，否则客户端会一直等待
        • 响应内容必须是字符串。不是的话就要用JSON.stringify()方法转化
        • 简写属性end方法（参数为发送的信息）
    • /表示首页

      Node中的js

  • Node为js提供了很多服务器级别的API，这些API绝大多数都被包装到了模块中，需要使用require进行引用

  • 当运行函数遇到require就会跳转到对应的模块中执行相应的代码，然后回来

  • 相对路径要加./

  • 在Node中没有全局作用域，只有模块作用域

  • 可以省略后缀名，默认.js

  • require（）方法的两个作用：

    • 拿到文件模块并执行里面的代码；
    • 拿到被加载文件模块到处的接口对象
